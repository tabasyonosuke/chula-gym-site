# CHULA GYM 綾瀬店 Webサイト

## 実装・対策概要

Webサイトの信頼性と検索エンジンへの最適化、およびユーザーの安全性を重視して構築

### SEO対策
* **適切なタイトルタグの配置**: 検索エンジンが重要視する `<title>` タグに、「店舗名（CHULA GYM）」「地域名（綾瀬店）」「業種（パーソナルトレーニングジム）」を網羅して記述
* **見出しタグ（hタグ）による論理構造**: `<h1>` でメインタイトルを定義し、`<h2>` や `<h3>` を用いて、ページの内容（選ばれる理由、料金、トレーナーなど）が検索エンジンに伝わりやすい階層構造に整理
* **レスポンシブデザインの viewport 設定**: `<meta name="viewport" content="width=device-width, initial-scale=1.0">` を記述し、スマホでの表示最適化な設計
* **言語指定**: `html lang="ja"` を指定し、検索エンジンに対してページが日本語で記述されていることを正しく伝えている

### セキュリティ対策
* **外部サイトへのリンク時の保護**: 公式LINEやGoogleマップなど、外部サイトへのリンクに対して `target="_blank"` とともに `rel="noopener noreferrer"` を付与しており、リンク先のサイトから元のページが操作される「タブハイジャック」を防ぐ対策
* **静的HTMLによる堅牢性**: サーバーサイドのプログラム（PHPやデータベースへの直接接続など）をHTML内に含めない静的なページ構成であるため、SQLインジェクションなどの一般的な攻撃リスクが低い安全な設計
